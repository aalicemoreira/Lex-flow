# ⚖️🎨 Design System - LexFlow

Neste projeto, utilizamos um framework UI focado em produtividade jurídica e aplicamos customizações minimalistas para refletir a identidade visual "Black & White".

### 1. Framework Base

- **Framework escolhido:** MaterializeCSS
- **Motivação:** Oferece componentes baseados no Material Design, permitindo uma prototipagem rápida e um visual corporativo, limpo e confiável para a gestão de leads advocatícios.

### 2. Paleta de Cores (Customização)

As variáveis de cor do framework foram escolhidas para transmitir sobriedade, autoridade e foco total na informação:

- **Cor Primária (Destaques/Ação):** #000000 (Preto absoluto)
  - _Uso:_ Botões de ação principal (Salvar, Novo Lead), ícones de status ativos e cabeçalho da barra de navegação.
- **Cor Secundária (Base):** 
#FFFFFF (Branco puro)
  - _Uso:_ Fundo das páginas, cards de leads e campos de input para garantir contraste máximo.
- **Cor de Fundo (Superfície):** #F9F9F9 (Cinza ultraclaro)
  - _Uso:_Fundo do Dashboard para destacar os painéis brancos e o painel lateral (Side Panel).
- **Cor de Neutra: (Bordas/Textos)** 
#E0E0E0 (Cinza médio)
  - _Uso:_ Divisores de tabela e estados inativos de botões de classificação.

### 3. Tipografia

Importada via Google Fonts para garantir uma leitura técnica e moderna:

- **Títulos (H1 a H6) e Branding:** `Poppins, sem-serifa` (Peso: 600/700).
- **Textos Corridos, Inputs e Tabelas de Leads:** `Poppins, sem-serifaf` (Peso: 400).
